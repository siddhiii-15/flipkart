# Flipkart Clone (MERN Stack)

A full-stack Flipkart inspired e-commerce application built using the MERN stack.

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
* Material UI

## Features

* Product listing page
* Product detail page
* Shopping cart
* Search functionality
* Responsive UI

## Personal Modifications

* Added custom welcome banner
* Personalized UI components

## Run the Project

Server

cd server
npm install
npm start

Client

cd client
npm install --legacy-peer-deps
npm start
